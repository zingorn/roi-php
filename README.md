# roi-php
ROI Test Problem

# Install
* Download Composer

```
php -r "readfile('https://getcomposer.org/installer');" | php
```

* Make project 

```
php composer.phar install
```

#Run tests

```
vendor/bin/phpunit tests/MarsRoverTest.php
```

or

```
php vendor/phpunit/phpunit/phpunit tests/MarsRoverTest.php
```
